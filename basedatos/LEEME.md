# Parte de base de datos del trabajo

Esta carpeta es exclusiva para Pilar, de base de datos. La carpeta consta de 4 partes:
1. Carpeta esquemas: en esta carpeta encontrarás capturas del entidad-relación y del lógico-relacional, aparte del archivo .dmd(datamodeler) con el que podrás comprobar la realización del estos.
2. Fichero "ficheroCargaDatos.sql", el cual contiene la creación de tablas y sus restricciones, la inserción de datos en dichas tablas y la creación de una vista por cada tabla que hay, en este caso 4. 
3. Fichero "procedimientoyfunciones.sql", el cual contiene el procedimiento y función que nos pedía el alta_compra y obtener_identificador_compra
4. Fichero "VISTAS.sql", el cual compra_por_codigo_postal y obtener_identificador_compra. 

Hemos realizado este proyecto utilizando una base de datos Mariadb, por lo que el tipo de las columnas de las tablas es acorde a dicha base de datos, por lo que para la comprobación del funcionamiento de esto debes utilizar un cliente que te permita conectar a mariadb.

Esperamos que te resulte sencillo revisarlo.
